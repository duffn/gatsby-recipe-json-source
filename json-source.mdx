# Source Content from JSON Within Gatsby

Setup Gatsby to source data from JSON files directly into a page or component.

This recipe installs and configures all the necessary NPM packages and Gatsby plugins to enable sourcing of content from JSON files.

---

## Install NPM Packages

<NPMPackage name="gatsby-source-filesystem" />

<NPMPackage name="gatsby-transformer-json" />

---

# Install Gatsby Plugins

First `gatsby-source-filesystem` to point to our JSON files which we'll keep in the `/src/data` directory.

<GatsbyPlugin
  name="gatsby-source-filesystem"
  key="src/data"
  options={{
    path: `./src/data/`,
  }}
/>

---

Then the `gatsby-transformer-json` plugin.

<GatsbyPlugin name="gatsby-transformer-json" />

---
